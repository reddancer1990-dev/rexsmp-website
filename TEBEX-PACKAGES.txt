REX SMP — TEBEX PACKAGES (copy into creator.tebex.io)
=====================================================

Before creating packages:
  1. Server console: tebex secret YOUR_KEY  (already done if /tebex works)
  2. Tebex -> Packages -> Create Package
  3. Delivery type: GAME SERVER COMMANDS (runs as console)
  4. Always use {username} — Tebex replaces with buyer's Minecraft name

Test each package once in TEST MODE before going live.


--- RANKS (Category: Ranks) ---

Package: Rex+ (30 Days)
  Price: set yourself (e.g. $4.99)
  Type: Subscription / 30 days if Tebex offers it
  Commands on purchase:
    lp user {username} parent addtemp rexplus 30d
  Commands on expiry (if subscription):
    lp user {username} parent remove rexplus

Package: Rex++ (30 Days)
  Price: set yourself (e.g. $9.99)
  Commands on purchase:

--- IF A COMMAND FAILS ---

Check in-game as OP first (replace YourName):
  points give YourName 100
  crates key give YourName common 1
  lp user YourName parent addtemp rexplus 30d

If keys fail, try:
  excellentcrates key give {username} common 1

Tebex dashboard -> Payments -> Queue -> retry failed commands
