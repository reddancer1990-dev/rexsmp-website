# Vault Notes

Obsidian-style notes app for iPhone and PC. Markdown, wiki links, graph view, tags, and offline PWA support.

## iPhone — test now

1. Open **https://reddancer1990-dev.github.io/rexsmp-website/** in Safari
2. Tap **Share** → **Add to Home Screen** for full-screen app experience
3. Notes save locally on your device (IndexedDB)

## Features

- Markdown editor with live preview
- `[[Wiki links]]` between notes (tap to navigate)
- Backlinks panel
- `#tags` with search
- Graph view of note connections
- Folders and file tree
- Export / import vault (sync to PC later)

## PC

Same URL works in any browser. Use Export vault on iPhone → Import vault on PC to move notes between devices.

## Local dev

```bash
npm install
npm run dev
```

Open the network URL on your iPhone (same Wi‑Fi) to test locally.
