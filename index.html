How To Join

Pick your platform and jump in

Java Edition
  1. Open Minecraft Java (1.20+)
  2. Click Multiplayer → Add Server
  3. Paste: rexsmp.simpleminecrafthosting.com
  4. Join and type /rtp to start exploring
Bedrock Edition
  1. Open Minecraft Bedrock / Pocket Edition
  2. Go to Play → Servers → Add Server
  3. Server: 69.41.172.83 · Port: 25691
  4. Save and connect — same world as Java players
