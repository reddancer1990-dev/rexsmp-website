How To Join

Same world on Java and Bedrock

Java Edition
  1. Open Minecraft Java (1.20+)
  2. Click Multiplayer → Add Server
  3. Server address: rexsmp.simpleminecrafthosting.com:33751
  4. Join and type /rtp to start exploring
Bedrock Edition
  1. Open Minecraft Bedrock / Pocket Edition
  2. Go to Play → Servers → Add Server
  3. Server: rexsmp.simpleminecrafthosting.com
  4. Port: 25691 — save and connect

Features

Built for PvP, economy, and crossplay

🏴

Base Raiding

Raiding is allowed. No land claims — protect your base with traps, not plugins.

💰

Dual Economy

/shop for money, /rexshop for RexShards, /sell and /worth to grind cash.

📦

Auction House

Buy and sell with /ah. Player orders with /orders.

🎁

Crate Keys

Common, Rare, and Legendary crates at spawn. Open with /crates or the hub.

⚡

Random Teleport

/rtp sends you 500–5000 blocks out. Find a spot and build your empire.

📊

Player Stats

/stats shows kills, deaths, money, RexShards, playtime, rank, and team.

📞

Calls & Friends

/call other players, /friend list, and proximity /voicechat.

🎵

Rexify Music

/airpods opens the Rexify music player — in-game tunes while you grind.

🔮

Rex Tools

RexDrill, RexSellAxe, amethyst gear, custom sidebar, and /settings GUI.

🤝

Rexford Trades

/rexford daily trades for bonus loot and RexShards deals.

🎲

Coin Flip PvP

Gamble money with /cf — create or join open coin flips against other players.

👥

Teams & Duels

/team for squads, /duels for 1v1 at spawn, /hub for warp menus.
